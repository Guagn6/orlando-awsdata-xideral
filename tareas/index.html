Estructura de los Datos

El dataset incluye las siguientes columnas principales:

Identificación

  • show_id: ID único del contenido
  • type: Movie o TV Show
  • title: Título del contenido

Metadatos

  • director: Director(es) del contenido
  • cast: Reparto principal
  • country: País de origen
  • rating: Clasificación por edades

Fechas y Duración

  • date_added: Fecha de incorporación
  • release_year: Año de estreno
  • duration: Duración (min o temporadas)

Clasificación

  • listed_in: Géneros y categorías
  • description: Sinopsis del contenido

Métodos de pandas Utilizados

.head() - Exploración Inicial

Muestra las primeras filas del DataFrame. Recibe un int que indica el número de filas por mostrar. Si no recibe argumento, muestra 5 filas por defecto. 

df.head(3)  # Muestra las primeras 3 filas

.isna() y .sum() - Detección de Valores Faltantes

Detecta valores faltantes (NaN) y cuenta la cantidad total de valores nulos por columna. 

df["title"].isna().sum()  # Cuenta valores nulos en la columna title

.value_counts() - Conteo de Frecuencias

Cuenta la frecuencia de valores únicos en una Series. Útil para análisis categórico. 

df["type"].value_counts()  # Cuenta Movies vs TV Shows

.groupby() y .mean() - Análisis por Grupos

Agrupa datos por una o más columnas y aplica funciones de agregación como promedio. 

df.groupby("type")["duration"].mean()  # Promedio de duración por tipo

.loc[] - Selección Condicional

Selecciona y asigna datos dentro de un DataFrame por filas y/o columnas usando condiciones. 

df.loc[df["type"] == "Movie", "duration"].mean()  # Promedio de películas

Análisis de Duración: Dos Métodos

Método 1: GroupBy

Utilizando groupby() para agrupar por tipo de contenido y calcular promedios. 

# Limpiar datos de duración
df["duration"] = df["duration"].str.replace(r'[^0-9]', '', regex=True).astype(float)

# Agrupar y calcular promedio
df.groupby("type")["duration"].mean()

Método 2: Filtrado con .loc[]

Usando .loc[] para filtrar y calcular promedios por separado. 

# Filtrar y calcular por separado
avgMovie = df.loc[df["type"] == "Movie", "duration"].mean()
avgTvShow = df.loc[df["type"] == "TV Show", "duration"].mean()

round(avgMovie, 0)  # ~100 minutos
round(avgTvShow, 0)  # ~2 temporadas
